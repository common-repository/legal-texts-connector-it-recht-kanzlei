Sichert Ihren Internetauftritt mit stets aktuellen und abmahnsicheren Rechtstexten der IT-Recht Kanzlei nach Buchung des AGB-Service und einer einmaligen Einrichtung ab.


== Description ==
Über die **Rechtstexte Schnittstelle der IT-Recht Kanzlei** werden Ihre abmahnsicheren Rechtstexte automatisch aus dem Mandantenportal auf Ihre Website übertragen.

Nachdem Sie einmalig die Verbindung zu Ihrem Account bei der IT-Recht Kanzlei hergestellt haben, haben Sie die Möglichkeit Ihre Rechtstexte dem Plugin via Schieberegler zuzuweisen. Mittels Shortcodes können Sie dann die übertragenen Rechtstexte jederzeit ganz einfach in beliebige Wordpress-Seiten einfügen.

Die Nutzung des Plugins setzt einen Zugang zum Mandantenportal der IT-Recht Kanzlei München voraus. Falls Sie noch kein Mandant sind, finden Sie [hier eine Möglichkeit das für Sie bedarfsgerechte Schutzpaket zu buchen](https://www.it-recht-kanzlei.de/schutzpakete.html).

= Features =
* Automatische Übertragung der Rechtstexte von der "IT-Recht Kanzlei München" in Ihre WordPress-Installation
* Automatische Aktualisierung der Rechtstexte in WordPress bei Bedarf
* Einfaches Einfügen von Rechtstexten auf Ihren Wordpress Seiten mittels Shortcodes
* Unabhängig von Ihrer Wordpress-Nutzung geeignet, sowohl für Shops (z.B. WooCommerce) als auch für WordPress-Websites/Blogs
* Hängt notwendige Rechtstexte als PDF an die WooCommerce-Email "Neue Bestellung" an
* Fremdsprachige Rechtstexte werden unterstützt
* Einfache Einrichtung (Stellt mittels Ihrer Zugangsdaten zum Mandantenportal der "IT-Recht Kanzlei München" eine Verbindung zur AGB-Schnittstelle her)

= Vorteile =
* Immer rechtlich aktuelle und abmahnsichere Rechtstexte für Ihren Online-Auftritt
* Die Rechtstexte-Schnittstelle liefert automatisch Updates der Rechtstexte
* Selbstverständlich: Anwaltliche Haftung (wie andere Rechtsanwaltskanzleien haftet auch die IT-Recht Kanzlei im Rahmen der gesetzlichen Bestimmungen für die Abmahnsicherheit der Rechtstexte)
* Der AGB-Service ist bei Bedarf jederzeit zum Monatsende kündbar
* Die Zahlung erfolgt auf Wunsch monatlich, vierteljährlich oder jährlich

== Installation ==
Navigieren Sie im WordPress Admin Bereich auf *Plugins => Neu hinzufügen*. Suchen Sie nach der **Rechtstexte-Schnittstelle der IT-Recht Kanzlei**. Klicken Sie auf *Jetzt installieren* und anschließend auf *Aktivieren*. Bitte beachten Sie, um das Plugin nutzen zu können benötigen Sie einen Zugang zum Mandantenportal der "IT-Recht Kanzlei München".

Wenn Sie bereits Mandant sind und noch keine AGB-Schnittstelle zu Wordpress eingerichtet haben, brauchen Sie sich nach der Aktivierung des Plugins nur mit Ihren Zugangsdaten zum Mandantenportal in der dann sichtbaren Login-Maske anmelden und Ihre Verbindung wird automatisch eingerichtet.

Falls Sie bereits das Plugin **AGB-Connector der IT-Recht Kanzlei** einsetzen, werden alle Konfigurationen des Plugins automatisch auf dieses Plugin übernommen.


== Frequently Asked Questions ==
Hier finden Sie Antworten auf häufig gestellte Fragen zu diesem Plugin.

= Kann ich die Rechtstexte-Schnittstelle auch ohne Zugang zum Mandantenportal der "IT-Recht Kanzlei München" nutzen? =
Nein, die Übertragung von Rechtstexten auf Ihre Wordpress-Präsenz inklusive automatischer Aktualisierung der Texte setzt die Nutzung eines der [Schutzpakete der IT-Recht Kanzlei München](https://www.it-recht-kanzlei.de/schutzpakete.html) voraus.

= Kann ich das Plugin "Rechtstexte-Schnittstelle der IT-Recht Kanzlei" sowohl für Wordpress-Shops als auch für Websites/Blogs ohne Verkaufsfunktion verwenden? =
Ja. Das Plugin unterstützt beide Arten uneingeschränkt. Es müssen nach der Installation lediglich die Shortcodes der gewünschten Rechtstexte aus der Plugin Einstellungsseite auf die entsprechende Seite(n) Ihrer Wordpress-Installation eingebunden und gespeichert werden.

= Anhängen der Rechtstexte an WooCommerce-E-Mails =
Bei Bestellungen in einem WooCommerce Shop werden die notwendigen Rechtstexte automatisch in der entsprechenden Sprache an die Bestellbestätigungen angehängt. Hierfür ist keine weitere Einrichtung Ihrerseits erforderlich.
